# ClawMama Skills

Official ClawMama skills repository.

## Skills

- `gaokao-volunteer-planner` — 高考志愿填报辅助决策：收集省份、分数、性别、家庭情况与偏好，生成院校、专业、城市、冲稳保匹配和最终报告。

## Install / use

```bash
npx skills add clawmama-run/clamama-skills --skill gaokao-volunteer-planner -y
npx skills use clawmama-run/clamama-skills@gaokao-volunteer-planner
```
